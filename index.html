Whole-Life Carbon framework (target architecture)

This calculator is being aligned with a modular whole-life-carbon workflow, split into site inputs, system sizing, demand, scenarios, and lifecycle reporting stages (A1-A5, B1-B7, C1-C4, D). The current UI covers the core PV energy model; additional modules are planned below.

Site + system inputs Now

  • Surface tilt + azimuth
  • PV capacity + PR
  • Reference year

  • Demand profiles
  • PV degradation
  • BESS dispatch

Embodied lifecycle Next
